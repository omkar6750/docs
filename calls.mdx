{/* How It Works */}
<div className="mb-12">
  <h2 className="text-2xl font-semibold text-gray-700 dark:text-white mb-4">How It Works</h2>
  
  ```mermaid
  sequenceDiagram
      participant App
      participant CometChatCalls
      participant MediaServer
      
      App->>CometChatCalls: 1. init(appId, region)
      App->>CometChatCalls: 2. login(authToken)
      App->>CometChatCalls: 3. generateToken(sessionId)
      App->>CometChatCalls: 4. joinSession(token, settings)
      CometChatCalls->>MediaServer: Connect to call
      MediaServer-->>CometChatCalls: Media streams
      CometChatCalls-->>App: Session joined + UI rendered
      App->>CometChatCalls: 5. Actions (mute, layout, etc.)
      CometChatCalls-->>App: Event callbacks
      App->>CometChatCalls: 6. leaveSession()
  ```
</div>
