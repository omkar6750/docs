## When to Use Advanced Features

| Scenario | Recommended Feature |
|----------|---------------------|
| VoIP app needing background connection | Manual mode + Background Keep-Alive |
| Custom user profiles | Custom Metadata |
| Filtering users by department | Tags & Filtering |
| Release build crashes | ProGuard Configuration |
| Push notifications | Push Notification Tokens |
| Optimizing performance | Best Practices |
