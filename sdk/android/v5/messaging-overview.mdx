Messaging is one of the core features of CometChat. We've thoughtfully created methods to help you send, receive and fetch message history.

At the minimum, you must add code for [sending messages](/sdk/android/v5/send-message) and [receiving messages](/sdk/android/v5/receive-messages).

Once you've implemented that, you can proceed to more advanced features like [typing indicators](/sdk/android/v5/typing-indicators) and [delivery & read receipts](/sdk/android/v5/delivery-read-receipts).
