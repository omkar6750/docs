<Warning>
**Legacy Version**: Android SDK v3 is no longer actively maintained. We strongly recommend migrating to [v4](/sdk/android/getting-started/overview) for the latest features, security updates, and support.
</Warning>

## Migration to v4

Please follow our [Migration Guide](/sdk/android/resources/migration-from-v3) to upgrade your application to the latest version.

### Why Upgrade?

| Feature | v3 | v4 |
|---------|----|----|
| Active Support | ❌ Limited | ✅ Full |
| New Features | ❌ No | ✅ Yes |
| Security Updates | ⚠️ Critical only | ✅ Regular |
| Performance | Good | Better |
| AI Features | ❌ No | ✅ Yes |

## v3 Documentation

If you need to reference v3 documentation for maintenance purposes, the original documentation is available in the archived section.

<Note>
For new projects, always use [Android SDK v4](/sdk/android/getting-started/overview).
</Note>

## Support

For v3-specific issues, please contact [CometChat Support](https://help.cometchat.com) with details about your migration timeline.
