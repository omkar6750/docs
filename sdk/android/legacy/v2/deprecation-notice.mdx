<Warning>
**Deprecated**: Android SDK v2 is deprecated and no longer supported. Please migrate to [v4](/sdk/android/getting-started/overview) immediately.
</Warning>

## End of Support

Android SDK v2 reached end-of-life and is no longer receiving:
- Security updates
- Bug fixes
- New features
- Technical support

## Migration Path

1. **v2 → v4**: Direct migration is recommended
2. Follow the [Migration Guide](/sdk/android/resources/migration-from-v3) (applicable for v2 as well)

## Immediate Action Required

If your application is still using v2:

1. Plan your migration timeline
2. Review the [v4 documentation](/sdk/android/getting-started/overview)
3. Test thoroughly in a development environment
4. Deploy the updated application

<Note>
Contact [CometChat Support](https://help.cometchat.com) if you need assistance with migration planning.
</Note>
