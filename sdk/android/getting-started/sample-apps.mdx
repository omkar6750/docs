## Official Sample Apps

### Kotlin Chat App (Recommended)

A complete chat application built with Kotlin showcasing modern Android development practices.

<Card title="Kotlin Chat App" icon="github" href="https://github.com/cometchat-pro/android-kotlin-chat-app">
  Full-featured chat app with 1-on-1 messaging, group chat, and calling
</Card>

**Features:**
- One-on-one messaging
- Group conversations
- Voice and video calling
- Message reactions
- Typing indicators
- Read receipts
- User presence

**Quick Start:**
```bash
git clone https://github.com/cometchat-pro/android-kotlin-chat-app.git
cd android-kotlin-chat-app
```

1. Open in Android Studio
2. Add your credentials in `Constants.kt`
3. Run on device or emulator

## Configuration

All sample apps require your CometChat credentials:

| Credential | Where to Find |
|------------|---------------|
| App ID | CometChat Dashboard → App → Overview |
| Auth Key | CometChat Dashboard → App → API Keys |
| Region | CometChat Dashboard → App → Overview (`us` or `eu`) |

<Warning>
**Security Note**: Auth Keys are for development only. Use [Auth Tokens](/sdk/android/authentication/login-with-auth-token) in production.
</Warning>

## Running on Emulator vs Device

### Emulator
- Works for messaging features
- Voice/video calling may have limitations
- Use x86 images for better performance

### Physical Device (Recommended)
- Full feature support
- Better performance for calling
- Real push notification testing

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Build fails | Ensure you have the latest Android Studio and Gradle |
| Login fails | Verify App ID, Auth Key, and Region |
| Messages not sending | Check internet connectivity |
| Calls not working | Ensure Calling SDK is included and permissions granted |
