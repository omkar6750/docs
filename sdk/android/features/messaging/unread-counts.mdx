Retrieve unread message counts to display badge indicators and keep users informed about new messages they haven't read yet.

## Overview

CometChat provides several methods to get unread message counts:

| Method | Description |
|--------|-------------|
| `getUnreadMessageCount()` | Total unread count across all conversations |
| `getUnreadMessageCountForUser()` | Unread count for a specific user conversation |
| `getUnreadMessageCountForGroup()` | Unread count for a specific group conversation |
| `getUnreadMessageCountForAllUsers()` | Unread counts for all user conversations |
| `getUnreadMessageCountForAllGroups()` | Unread counts for all group conversations |

## Get Total Unread Count

Get the total number of unread messages across all conversations:

<Tabs>
<Tab title="Kotlin">
```kotlin
CometChat.getUnreadMessageCount(object : CometChat.CallbackListener<HashMap<String, Int>>() {
    override fun onSuccess(counts: HashMap<String, Int>) {
        val userCount = counts["user"] ?: 0
        val groupCount = counts["group"] ?: 0
        val totalCount = userCount + groupCount
        
        Log.d("UnreadCount", "Users: $userCount, Groups: $groupCount, Total: $totalCount")
        updateBadge(totalCount)
    }

    override fun onError(e: CometChatException) {
        Log.e("UnreadCount", "Error: ${e.message}")
    }
})
```
</Tab>
<Tab title="Java">
```java
CometChat.getUnreadMessageCount(new CometChat.CallbackListener<HashMap<String, Integer>>() {
    @Override
    public void onSuccess(HashMap<String, Integer> counts) {
        int userCount = counts.get("user") != null ? counts.get("user") : 0;
        int groupCount = counts.get("group") != null ? counts.get("group") : 0;
        int totalCount = userCount + groupCount;
        
        Log.d("UnreadCount", "Users: " + userCount + ", Groups: " + groupCount + ", Total: " + totalCount);
        updateBadge(totalCount);
    }

    @Override
    public void onError(CometChatException e) {
        Log.e("UnreadCount", "Error: " + e.getMessage());
    }
});
```
</Tab>
</Tabs>

### Response Format

The response is a `HashMap<String, Int>` with keys:
- `"user"` - Total unread messages from user conversations
- `"group"` - Total unread messages from group conversations

Get unread message count for a specific user conversation:

<Tabs>
<Tab title="Kotlin">
```kotlin
val uid = "user_123"

CometChat.getUnreadMessageCountForUser(uid, object : CometChat.CallbackListener<HashMap<String, Int>>() {

        Log.e("UnreadCount", "Error: ${e.message}")
    }
```

</Tab>

```java
CometChat.getUnreadMessageCountForUser(uid, new CometChat.CallbackListener<HashMap<String, Integer>>() {

    public void onError(CometChatException e) {
```

## Get Unread Counts for All Users

Get unread message counts for all user conversations at once:

<Tabs>
<Tab title="Kotlin">
```kotlin
CometChat.getUnreadMessageCountForAllUsers(object : CometChat.CallbackListener<HashMap<String, Int>>() {
    override fun onSuccess(counts: HashMap<String, Int>) {
        counts.forEach { (uid, count) ->
            Log.d("UnreadCount", "User $uid: $count unread")
        }
    }

    override fun onError(e: CometChatException) {
        Log.e("UnreadCount", "Error: ${e.message}")
    }
})
```
</Tab>
<Tab title="Java">
```java
CometChat.getUnreadMessageCountForAllUsers(new CometChat.CallbackListener<HashMap<String, Integer>>() {
    @Override
    public void onSuccess(HashMap<String, Integer> counts) {
        for (Map.Entry<String, Integer> entry : counts.entrySet()) {
            Log.d("UnreadCount", "User " + entry.getKey() + ": " + entry.getValue() + " unread");
        }
    }

    @Override
    public void onError(CometChatException e) {
        Log.e("UnreadCount", "Error: " + e.getMessage());
    }
});
```
</Tab>
</Tabs>

## Hide Messages from Blocked Users

By default, messages from blocked users are included in unread counts. To exclude them:

<Tabs>
<Tab title="Kotlin">
```kotlin
val hideMessagesFromBlockedUsers = true

CometChat.getUnreadMessageCount(
    hideMessagesFromBlockedUsers,
    object : CometChat.CallbackListener<HashMap<String, Int>>() {
        override fun onSuccess(counts: HashMap<String, Int>) {
            // Counts exclude messages from blocked users
            Log.d("UnreadCount", "Counts: $counts")
        }

        override fun onError(e: CometChatException) {
            Log.e("UnreadCount", "Error: ${e.message}")
        }
    }
)
```
</Tab>
<Tab title="Java">
```java
boolean hideMessagesFromBlockedUsers = true;

CometChat.getUnreadMessageCount(
    hideMessagesFromBlockedUsers,
    new CometChat.CallbackListener<HashMap<String, Integer>>() {
        @Override
        public void onSuccess(HashMap<String, Integer> counts) {
            // Counts exclude messages from blocked users
            Log.d("UnreadCount", "Counts: " + counts);
        }

        @Override
        public void onError(CometChatException e) {
            Log.e("UnreadCount", "Error: " + e.getMessage());
        }
    }
);
```
</Tab>
</Tabs>

## Implementing Badge UI

Example of updating an app badge with unread count:

<Tabs>
<Tab title="Kotlin">
```kotlin
class ChatBadgeManager(private val context: Context) {
    
    fun updateBadge() {
        CometChat.getUnreadMessageCount(object : CometChat.CallbackListener<HashMap<String, Int>>() {
            override fun onSuccess(counts: HashMap<String, Int>) {
                val total = (counts["user"] ?: 0) + (counts["group"] ?: 0)
                setBadgeCount(total)
            }

            override fun onError(e: CometChatException) {
                Log.e("Badge", "Failed to get count: ${e.message}")
            }
        })
    }
    
    private fun setBadgeCount(count: Int) {
        // Using ShortcutBadger library or similar
        try {
            ShortcutBadger.applyCount(context, count)
        } catch (e: Exception) {
            Log.e("Badge", "Badge not supported: ${e.message}")
        }
    }
}
```
</Tab>
<Tab title="Java">
```java
public class ChatBadgeManager {
    private Context context;
    
    public ChatBadgeManager(Context context) {
        this.context = context;
    }
    
    public void updateBadge() {
        CometChat.getUnreadMessageCount(new CometChat.CallbackListener<HashMap<String, Integer>>() {
            @Override
            public void onSuccess(HashMap<String, Integer> counts) {
                int userCount = counts.get("user") != null ? counts.get("user") : 0;
                int groupCount = counts.get("group") != null ? counts.get("group") : 0;
                int total = userCount + groupCount;
                setBadgeCount(total);
            }

            @Override
            public void onError(CometChatException e) {
                Log.e("Badge", "Failed to get count: " + e.getMessage());
            }
        });
    }
    
    private void setBadgeCount(int count) {
        try {
            ShortcutBadger.applyCount(context, count);
        } catch (Exception e) {
            Log.e("Badge", "Badge not supported: " + e.getMessage());
        }
    }
}
```
</Tab>
</Tabs>

## Best Practices

1. **Cache counts locally** - Don't fetch on every screen transition
2. **Update on message events** - Refresh counts when receiving new messages
3. **Use listeners** - Update counts in real-time using message listeners
4. **Handle errors gracefully** - Show cached count if fetch fails
