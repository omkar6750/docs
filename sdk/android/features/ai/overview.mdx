CometChat provides AI-powered features to enhance your chat experience. These features leverage machine learning to provide intelligent assistance to users.

## Available AI Features

| Feature | Description | Use Case |
|---------|-------------|----------|
| [Smart Replies](/sdk/android/features/ai/smart-replies) | AI-generated reply suggestions | Quick responses |
| [Conversation Summary](/sdk/android/features/ai/conversation-summary) | Summarize long conversations | Catch up on missed messages |
| [Ask Bot](/sdk/android/features/ai/ask-bot) | Query AI bots | Customer support, FAQ |
| [AI Moderation](/sdk/android/features/ai/ai-moderation) | Content moderation | Filter inappropriate content |
| [AI Agents](/sdk/android/features/ai/ai-agents) | Automated AI agents | Automated responses |
| [AI Chatbots](/sdk/android/features/ai/ai-chatbots) | Conversational bots | Interactive assistance |
| [AI Copilot](/sdk/android/features/ai/ai-user-copilot) | User assistance | Writing help |

## Quick Start

### Get Smart Replies

<Tabs>
<Tab title="Kotlin">
```kotlin
import com.cometchat.chat.core.CometChat
import com.cometchat.chat.exceptions.CometChatException

val receiverId = "user123"
val receiverType = CometChatConstants.RECEIVER_TYPE_USER

CometChat.getSmartReplies(receiverId, receiverType, object : CometChat.CallbackListener<HashMap<String, String>>() {
    override fun onSuccess(replies: HashMap<String, String>) {
        replies.forEach { (key, reply) ->
            Log.d("SmartReply", "$key: $reply")
        }
    }
    
    override fun onError(e: CometChatException) {
        Log.e("SmartReply", "Error: ${e.message}")
    }
})
```
</Tab>
<Tab title="Java">
```java
import com.cometchat.chat.core.CometChat;
import com.cometchat.chat.exceptions.CometChatException;

String receiverId = "user123";
String receiverType = CometChatConstants.RECEIVER_TYPE_USER;

CometChat.getSmartReplies(receiverId, receiverType, new CometChat.CallbackListener<HashMap<String, String>>() {
    @Override
    public void onSuccess(HashMap<String, String> replies) {
        for (Map.Entry<String, String> entry : replies.entrySet()) {
            Log.d("SmartReply", entry.getKey() + ": " + entry.getValue());
        }
    }
    
    @Override
    public void onError(CometChatException e) {
        Log.e("SmartReply", "Error: " + e.getMessage());
    }
});
```
</Tab>
</Tabs>

## Prerequisites

AI features require:
1. AI features enabled in your CometChat Dashboard
2. Appropriate subscription plan
3. SDK version 4.0+

<Note>
AI features may have usage limits based on your subscription plan. Check your dashboard for details.
</Note>
