## Summary

Messaging is one of the core features of CometChat. The Chat SDK provides comprehensive methods to send, receive, and fetch message history, enabling real-time communication between users and within groups.

## When to Use

- Building one-on-one chat functionality between users
- Implementing group messaging for communities or teams
- Adding real-time message delivery and read receipts
- Creating threaded conversations for organized discussions
- Implementing typing indicators for better user experience
- Supporting rich media messages (images, videos, files)
