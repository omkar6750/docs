{/* TL;DR for Agents and Quick Reference */}
<Info>
**Quick Setup Reference**

```bash
npm install @cometchat/calls-sdk-javascript
```

```javascript
import { CometChatCalls } from "@cometchat/calls-sdk-javascript";

const callAppSetting = new CometChatCalls.CallAppSettingsBuilder()
  .setAppId("APP_ID")
  .setRegion("REGION")
  .build();

CometChatCalls.init(callAppSetting).then(() => console.log("Calls SDK ready"));
```

**Required:** App ID, Region from [CometChat Dashboard](https://app.cometchat.com)
</Info>

<Warning>
`CometChatCalls.init()` must be called before any other Calls SDK method. Make sure the Chat SDK is also initialized via `CometChat.init()` first (unless using [Standalone Calling](/sdk/javascript/standalone-calling)).
</Warning>

### Initialize CometChatCalls

The `init()` method initializes the Calls SDK. It takes a `CallAppSettings` instance as its parameter.

`CallAppSettingsBuilder` accepts three settings:

| Setting | Description |
| ------- | ----------- |
| `appID` | Your CometChat App ID |
| `region` | The region where your app was created |
| `host` | (Optional) Custom client URL for dedicated deployments |

Call `init()` before any other `CometChatCalls` method — ideally on app startup.

<Tabs>
<Tab title="TypeScript">
```typescript
let appID = "APP_ID";
let region = "REGION";

const callAppSetting = new CometChatCalls.CallAppSettingsBuilder()
  .setAppId(appID)
  .setRegion(region)
  .build();

CometChatCalls.init(callAppSetting).then(
  () => {
    console.log("CometChatCalls initialization completed successfully");
  },
  (error) => {
    console.log("CometChatCalls initialization failed with error:", error);
  }
);
```

</Tab>

<Tab title="JavaScript">
```js
let appID = "APP_ID";
let region = "REGION";

const callAppSetting = new CometChatCalls.CallAppSettingsBuilder()
  .setAppId(appID)
  .setRegion(region)
  .build();

CometChatCalls.init(callAppSetting).then(
  () => {
    console.log("CometChatCalls initialization completed successfully");
  },
  (error) => {
    console.log("CometChatCalls initialization failed with error:", error);
  }
);
```

</Tab>

</Tabs>

Make sure you replace the `APP_ID` with your CometChat **App ID** and `REGION` with your **App Region** in the above code.

| Parameter         | Description                              |
| ----------------- | ---------------------------------------- |
| `callAppSettings` | An object of the `CallAppSettings` class |
