## Summary

CometChat's Calls SDK provides voice and video calling capabilities for your web application. The Calls SDK supports multiple implementation approaches including ringing flows with call notifications, direct call sessions, and standalone calling without the Chat SDK dependency.

## When to Use This

- Add voice and video calling to your chat application
- Implement incoming/outgoing call flows with accept/reject functionality
- Create direct call sessions for meetings or group calls
- Build standalone calling features without the Chat SDK
- Add call features like recording, screen sharing, and virtual backgrounds
