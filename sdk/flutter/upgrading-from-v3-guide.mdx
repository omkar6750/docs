<Accordion title="AI Integration Quick Reference">

Key changes from v3 to v4:
- Chat SDK Import: `import 'package:cometchat_sdk/cometchat_sdk.dart';`
- Calls SDK Import: `import 'package:cometchat_calls_sdk/cometchat_calls_sdk.dart';`
- Setup: Follow the updated [Setup Guide](/sdk/flutter/setup) for v4 installation
- Migration: Update `pubspec.yaml`, replace imports, run `flutter pub get`
</Accordion>

Upgrading from v3 to v4 is straightforward — the API surface is largely the same, only the package names and imports changed. Follow the [v4 setup instructions](/sdk/flutter/setup) first, then update your dependencies and imports as shown below.

## Update Dependencies

Update your `pubspec.yaml` to use the v4 SDK packages. Follow the [Setup](/sdk/flutter/setup) instructions to install the latest v4 version.

## Chat Import Change

Remove all the imports of CometChat Flutter v3 SDK or replace it with a single import statement:

<Tabs>
<Tab title="Dart">
```dart
import 'package:cometchat_sdk/cometchat_sdk.dart';
```
</Tab>
</Tabs>

## Calling Import Change

Remove all the imports of CometChat Flutter v3 Calls SDK or replace it with a single import statement:

<Tabs>
<Tab title="Dart">
```dart
import 'package:cometchat_calls_sdk/cometchat_calls_sdk.dart';
```
</Tab>
</Tabs>

## Migration Steps

1. Update `pubspec.yaml` with v4 SDK versions
2. Replace all v3 import statements with the new unified imports shown above
3. Run `flutter pub get` to install updated dependencies
4. Test your application for any breaking changes

<Note>
The API methods, class names, and listener interfaces are unchanged between v3 and v4. Once you update the packages and imports, your existing code should work without further modifications.
</Note>
