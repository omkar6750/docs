---
title: "Upgrading from v4"
---

This guide helps you migrate your Flutter application from CometChat SDK v4 to v5. It covers breaking changes, deprecations, and new features.

## Installation

Update your `pubspec.yaml` to use the v5 SDK:

<Tabs>
<Tab title="Dart">
```yaml
dependencies:
  cometchat_sdk: ^5.0.0
```
</Tab>
</Tabs>

Then run:
```bash
flutter pub get
```

## Breaking Changes

### Removed Deprecated Methods

The following deprecated methods have been removed in v5. You must update your code to use the new methods:

| Removed Method | Replacement |
|----------------|-------------|
| `markAsUnread()` | `markMessageAsUnread()` |
| `receaverUid` parameter | `receiverUid` (now required) |
| `fetchPushPreferences()` | `fetchPreferences()` |
| `updatePushPreferences()` | `updatePreferences()` |
| `resetPushPreferences()` | `resetPreferences()` |
| `PushPreferences` class | `NotificationPreferences` class |

### Default Limit Change

The default limit for fetching data in various request builders has changed from `50` to `30`. If your app relies on the default limit, you may need to explicitly set it:

```dart
MessagesRequest messageRequest = (MessagesRequestBuilder()
  ..uid = UID
  ..limit = 50  // Explicitly set if you need 50
).build();
```

## Method Migrations

### Mark as Unread

The `markAsUnread()` method has been removed. Use `markMessageAsUnread()` instead.

**Before (v4):**
```dart
CometChat.markAsUnread(message, onSuccess: ..., onError: ...);
```

**After (v5):**
```dart
CometChat.markMessageAsUnread(message, onSuccess: ..., onError: ...);
```

### Typing Indicator Parameter

The `receaverUid` parameter has been removed from `startTyping()` and `endTyping()` methods. Use `receiverUid` instead (now required).

**Before (v4):**
```dart
CometChat.startTyping(receaverUid: "UID", receiverType: ...);
```

**After (v5):**
```dart
CometChat.startTyping(receiverUid: "UID", receiverType: ...);
```

### Push Notification Preferences

The following methods and classes have been removed:

| Removed | Replacement |
|---------|-------------|
| `fetchPushPreferences()` | `fetchPreferences()` |
| `updatePushPreferences()` | `updatePreferences()` |
| `resetPushPreferences()` | `resetPreferences()` |
| `PushPreferences` class | `NotificationPreferences` class |

**Before (v4):**
```dart
final prefs = await CometChatNotifications.fetchPushPreferences();
```

**After (v5):**
```dart
final prefs = await CometChatNotifications.fetchPreferences();
```

## New Features

### Conversation Management

New methods for managing conversation status:

```dart
// Mark entire conversation as delivered
CometChat.markConversationAsDelivered(conversation, onSuccess: ..., onError: ...);

// Mark entire conversation as read
CometChat.markConversationAsRead(conversation, onSuccess: ..., onError: ...);
```

### Multiple Media Files

Send multiple media files in a single message:

```dart
List<String> filePaths = ["path/to/image1.jpg", "path/to/image2.jpg"];

MediaMessage mediaMessage = MediaMessage(
  receiverType: receiverType,
  type: messageType,
  receiverUid: receiverID,
  files: filePaths,
);

await CometChat.sendMediaMessage(mediaMessage, onSuccess: ..., onError: ...);
```

### Agentic Conversations

Filter AI-driven conversations:

```dart
// Hide agentic conversations
ConversationsRequest request = (ConversationsRequestBuilder()
  ..hideAgentic = true
).build();

// Show only agentic conversations
ConversationsRequest request = (ConversationsRequestBuilder()
  ..onlyAgentic = true
).build();
```

### User/Group Tags Filter

Filter conversations by user or group tags:

```dart
ConversationsRequest request = (ConversationsRequestBuilder()
  ..userTags = ["premium", "verified"]
  ..groupTags = ["support"]
).build();
```

### Enhanced User Search

New search and sort options for users:

```dart
UsersRequest usersRequest = (UsersRequestBuilder()
  ..searchKeyword = "john"
  ..searchIn = ["name", "uid"]
  ..sortBy = "name"
  ..sortByOrder = "asc"
).build();
```

### Create Group with Members

Create a group and add members in one step:

```dart
List<GroupMember> members = [
  GroupMember(uid: "uid1", scope: CometChatMemberScope.participant),
  GroupMember(uid: "uid2", scope: CometChatMemberScope.moderator),
];

await CometChat.createGroupWithMembers(
  group: group,
  members: members,
  onSuccess: (Group group, Map<String, String> failedMembers) { ... },
  onError: (CometChatException e) { ... }
);
```

### Group Member Status Filter

Filter group members by online/offline status:

```dart
GroupMembersRequest request = (GroupMembersRequestBuilder(GUID)
  ..status = CometChatUserStatus.online
).build();
```

### Direct Page Navigation

Jump to specific pages in paginated results:

```dart
MessagesRequest messageRequest = (MessagesRequestBuilder()
  ..uid = UID
  ..page = 2
  ..limit = 30
).build();
```

### Attachment Type Filter

Filter messages by attachment types:

```dart
MessagesRequest messageRequest = (MessagesRequestBuilder()
  ..uid = UID
  ..attachmentTypes = ["image", "video"]
).build();
```

### Hide Quoted Messages

Exclude quoted messages from response:

```dart
MessagesRequest messageRequest = (MessagesRequestBuilder()
  ..uid = UID
  ..hideQuotedMessages = true
).build();
```

## Migration Checklist

- [ ] Update SDK version in `pubspec.yaml`
- [ ] Run `flutter pub get`
- [ ] Replace `markAsUnread()` with `markMessageAsUnread()`
- [ ] Replace `receaverUid` with `receiverUid` in typing methods
- [ ] Update push notification preference methods (`fetchPushPreferences` → `fetchPreferences`, etc.)
- [ ] Replace `PushPreferences` class with `NotificationPreferences`
- [ ] Review default limit changes if relying on defaults
- [ ] Test all messaging and conversation features
- [ ] Test AI features if using AI agents or moderation
