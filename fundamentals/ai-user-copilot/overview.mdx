CometChat's Smart Chat Features bring AI-powered capabilities to your messaging experience. These features use OpenAI's ChatGPT to help users start conversations naturally, respond quickly, and stay on top of lengthy discussions.

## Conversation Starter

Conversation Starter suggests opening messages for a conversation, helping users break the ice and get the chat flowing. It generates contextually relevant suggestions based on the conversation participants and history.

[Learn more about Conversation Starter →](/fundamentals/ai-user-copilot/conversation-starter)

## Smart Replies

Smart Replies provides AI-generated response suggestions based on the context of the conversation. Users can select a suggested reply to respond quickly without typing a full message.

[Learn more about Smart Replies →](/fundamentals/ai-user-copilot/smart-replies)

## Conversation Summary

Conversation Summary generates a concise recap of a conversation, helping users catch up on discussions they may have missed. This is especially useful for long group conversations or when returning after being away.

[Learn more about Conversation Summary →](/fundamentals/ai-user-copilot/conversation-summary)

## Pre-requisite

* Login to your [CometChat dashboard](https://app.cometchat.com/login) and choose your app.
* Navigate to **AI Chatbot** > **Settings** in the left-hand menu.

### Set the GPT Model

Enter the name of the OpenAI ChatGPT model that you intend to use.

### Save the OpenAI Key

You can get the OpenAI Key from your [OpenAI Account](https://platform.openai.com/account/api-keys). This will be used by CometChat to interact with the OpenAI APIs.

### Set a Custom Instruction

Custom Instruction is information that gets added to each ChatGPT prompt made by CometChat AI. Use it to describe your use case and inform what kind of responses you need.

### Set the Temperature

The API is non-deterministic by default, meaning you might get slightly different completions each time. Setting temperature to 0 makes outputs mostly deterministic.

Lower values produce more consistent outputs, while higher values generate more diverse and creative results. Select a temperature value based on the desired trade-off between coherence and creativity for your application.

### Enable AI

Toggle on AI to activate Smart Chat Features.
