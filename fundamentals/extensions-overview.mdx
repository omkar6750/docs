Extensions extend the functionality of CometChat beyond core messaging. They add capabilities that enhance user experience, boost engagement, enable collaboration, and more. Extensions can be enabled from the [CometChat Dashboard](https://app.cometchat.com/login) under the Extensions section.

## User Experience

Extensions that improve the messaging experience for your users.

- **[Bitly](/fundamentals/bitly)** — Generate meta descriptions for URLs shared in conversations.
- **[Link Preview](/fundamentals/link-preview)** — Automatically generate rich link previews for URLs shared in messages.
- **[Message Shortcuts](/fundamentals/message-shortcuts)** — Send pre-defined messages with configurable shortcuts.
- **[Pin Message](/fundamentals/pin-message)** — Pin important messages in a conversation for quick access.
- **[Rich Media Preview](/fundamentals/rich-media-preview)** — Generate rich media previews for popular media links.
- **[Save Messages](/fundamentals/save-message)** — Enable users to save messages from one-on-one and group conversations.
- **[Thumbnail Generation](/fundamentals/thumbnail-generation)** — Generate thumbnails for images and videos shared in conversations.
- **[TinyURL](/fundamentals/tinyurl)** — Convert long website links into short, manageable URLs.
- **[Voice Transcription](/fundamentals/voice-transcription)** — Transcribe audio messages using speech-to-text.

## User Engagement

Extensions that help increase user engagement in your app.

- **[Giphy](/fundamentals/giphy)** — Search and share animated GIFs directly in conversations.
- **[Message Translation](/fundamentals/message-translation)** — Translate text messages in different languages on the fly.
- **[Polls](/fundamentals/polls)** — Quickly ask for opinions in chats with interactive polls.
- **[Reminders](/fundamentals/reminders)** — Create reminders for messages or anything else within conversations.
- **[Stickers](/fundamentals/stickers)** — Send and manage stickers in conversations.
- **[Stipop](/fundamentals/stickers-stipop)** — Create meaningful experiences in your app with stickers that users love.
- **[Tenor](/fundamentals/tenor)** — Bring personality to your conversations with Tenor GIFs.

## Collaboration

Extensions that enable collaboration within your app.

- **[Collaborative Document](/fundamentals/collaborative-document)** — Allow users to create and share documents for collaboration.
- **[Collaborative Whiteboard](/fundamentals/collaborative-whiteboard)** — Allow users to create and share whiteboards for collaboration.

## Customer Support

Extensions that help you add customer support to your app.

- **[Chatwoot](/fundamentals/chatwoot)** — Connect and exchange messages between CometChat and Chatwoot.
- **[Intercom](/fundamentals/intercom)** — Connect and exchange messages between CometChat and Intercom.

## Security

Extensions that add extra security to your messaging experience.

- **[Disappearing Messages](/fundamentals/disappearing-messages)** — Allow end users to send messages that disappear after a set interval or time.
