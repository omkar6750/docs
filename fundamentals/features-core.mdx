CometChat's core features provide the essential building blocks for a complete in-app messaging experience. These features are available out of the box with CometChat SDKs and UI Kits, requiring minimal configuration to get started.

## Instant Messaging

Instant messaging enables real-time one-on-one and group conversations. Messages are sent via CometChat SDKs, which act as an API wrapper, and delivered to recipients through a persistent WebSocket connection managed by the platform.

### Key Capabilities

- Real-time text messaging between two users or within a group
- Persistent message history with automatic syncing across devices
- WebSocket-based delivery for low-latency communication
- Offline message queuing and delivery upon reconnection

### Implementation

- [Send Messages](/sdk/javascript/send-message) — JavaScript SDK
- [Receive Messages](/sdk/javascript/receive-message) — JavaScript SDK
- [Managing WebSocket Connections](/sdk/javascript/managing-web-sockets-connections-manually) — JavaScript SDK

<Info>Instant messaging is supported out of the box in all CometChat UI Kits. Extend messaging with [Message Translation](/fundamentals/message-translation) and [Disappearing Messages](/fundamentals/disappearing-messages).</Info>

## Media Sharing

Media sharing allows users to send and receive images, videos, audio files, and documents within conversations. CometChat supports a variety of media types to keep conversations rich and engaging.

### Key Capabilities

- Share images, videos, audio files, and documents
- Automatic media preview and thumbnail generation
- Support for custom message types for specialized media
- File size and type validation

### Implementation

- [Send Media Messages](/sdk/javascript/send-message#media-message) — JavaScript SDK
- [Extensions](/fundamentals/extensions-overview) — GIFs and Stickers via extensions

<Info>Media sharing is supported out of the box in all CometChat UI Kits. Enhance media sharing with extensions like [Giphy](/fundamentals/giphy), [Stickers](/fundamentals/stickers), and [Tenor](/fundamentals/tenor).</Info>

## Read Receipts

Read receipts let users know when their messages have been delivered and read. CometChat provides delivery receipts (message reached the recipient) and read receipts (message was seen by the recipient).

### Key Capabilities

- Mark messages as delivered when they reach the recipient's device
- Mark messages as read when the recipient views them
- Real-time receipt updates via WebSocket events
- Support for both one-on-one and group conversations

### Implementation

- [Delivery & Read Receipts](/sdk/javascript/delivery-read-receipts) — JavaScript SDK

<Info>Read receipts are supported out of the box in all CometChat UI Kits.</Info>

## Mark as Unread

Mark as Unread allows users to flag a conversation or message as unread, helping them revisit important messages later. This is useful for tracking conversations that need follow-up.

### Key Capabilities

- Mark individual messages or entire conversations as unread
- Visual indicators for unread status in conversation lists
- Unread count tracking per conversation

### Implementation

- [Mark Messages as Unread](/sdk/javascript/delivery-read-receipts#mark-messages-as-unread) — JavaScript SDK

<Info>Mark as Unread is supported out of the box in all CometChat UI Kits.</Info>

## Typing Indicators

Typing indicators show real-time feedback when another user is composing a message. This creates a more natural conversational experience by letting participants know a response is on the way.

### Key Capabilities

- Real-time typing status broadcast to conversation participants
- Support for both one-on-one and group conversations
- Automatic timeout when a user stops typing

### Implementation

- [Typing Indicators](/sdk/javascript/typing-indicators) — JavaScript SDK

<Info>Typing indicators are supported out of the box in all CometChat UI Kits.</Info>

## User Presence

User presence displays the online or offline status of users, helping participants know who is currently available for conversation.

### Key Capabilities

- Real-time online/offline status updates
- Last active timestamp for offline users
- Subscribe to presence updates for specific users
- Custom status support

### Implementation

- [User Presence](/sdk/javascript/user-presence) — JavaScript SDK

<Info>User presence is supported out of the box in all CometChat UI Kits.</Info>

## Reactions

Reactions allow users to respond to individual messages with emojis, providing a quick and lightweight way to express acknowledgment or emotion without sending a full message.

### Key Capabilities

- React to any message with one or more emojis
- Multiple users can react to the same message
- Real-time reaction updates across all participants
- Toggle reactions on and off

### Implementation

- [Reactions](/sdk/javascript/reactions) — JavaScript SDK

<Info>Reactions are supported out of the box in all CometChat UI Kits.</Info>

## Mentions

Mentions let users tag specific participants in a conversation using the @ symbol, drawing their attention to a particular message. This is especially useful in group conversations.

### Key Capabilities

- Tag users with @username in messages
- Highlighted mention display in message bubbles
- Notification triggers for mentioned users
- Support for mentioning multiple users in a single message

### Implementation

- [Mentions](/sdk/javascript/mentions) — JavaScript SDK

<Info>Mentions are supported out of the box in all CometChat UI Kits. See also [Message Shortcuts](/fundamentals/message-shortcuts) for pre-defined quick replies.</Info>

## Threaded Conversations

Threaded conversations allow users to branch off from a specific message to create a focused sub-conversation. This keeps the main conversation clean while enabling detailed discussions on specific topics.

### Key Capabilities

- Reply to any message in a dedicated thread
- Thread reply counts visible on the parent message
- Independent notification settings for threads
- Full messaging capabilities within threads (media, reactions, etc.)

### Implementation

- [Threaded Messages](/sdk/javascript/threaded-messages) — JavaScript SDK

<Info>Threaded conversations are supported out of the box in all CometChat UI Kits.</Info>

## Quoted Replies

Quoted replies let users respond to a specific message by quoting it, providing clear context for the reply. The original message is displayed alongside the response so all participants can follow the conversation.

### Key Capabilities

- Quote any message (text, media, or custom) when replying
- Visual display of the quoted message within the reply bubble
- Navigate to the original message from the quote
- Support in both one-on-one and group conversations

### Implementation

- [Receive Messages](/sdk/javascript/receive-message) — JavaScript SDK

<Info>Quoted replies are supported out of the box in all CometChat UI Kits.</Info>

## Group Chats

Group chats enable multi-user conversations where members can communicate in real time. CometChat supports creating, managing, and participating in groups with flexible membership and permission controls.

### Key Capabilities

- Create public, private, and password-protected groups
- Add, remove, and manage group members
- Assign roles such as admin, moderator, and participant
- Group-level settings for notifications and permissions
- Transfer group ownership

### Implementation

- [Create a Group](/sdk/javascript/create-group) — JavaScript SDK
- [Join a Group](/sdk/javascript/join-group) — JavaScript SDK
- [Group Members](/sdk/javascript/retrieve-group-members) — JavaScript SDK

<Info>Group chats are supported out of the box in all CometChat UI Kits. Extend group collaboration with [Collaborative Document](/fundamentals/collaborative-document) and [Collaborative Whiteboard](/fundamentals/collaborative-whiteboard).</Info>

## Report Message

The Report Message feature allows users to flag inappropriate or harmful messages for review. This helps maintain a safe and respectful messaging environment by enabling moderation workflows.

### Key Capabilities

- Report any message within a conversation
- Attach a reason or category to the report
- Reported messages are flagged for admin or moderator review

### Implementation

- [Report Message](/sdk/javascript/edit-message#report-a-message) — JavaScript SDK

<Info>Report Message is supported out of the box in all CometChat UI Kits.</Info>

## Conversation and Advanced Search

Conversation and Advanced Search enables users to find specific conversations, messages, or users across the platform. Advanced filters allow narrowing results by user, group, message type, attachments, reactions, mentions, and more.

### Key Capabilities

- Search conversations by user or group name
- Search messages using advanced filters (text, attachments, reactions, mentions)
- Filter by specific message types or date ranges
- Full-text search across message history

### Implementation

- [Retrieve Conversations](/sdk/javascript/retrieve-conversations) — JavaScript SDK

<Info>Conversation and Advanced Search is supported out of the box in all CometChat UI Kits.</Info>
