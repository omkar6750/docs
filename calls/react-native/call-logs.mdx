CometChat's React Native Call SDK lets you integrate call logs into your app, giving users a clear view of their communication history. Call logs include details like call duration, participants, and more.

You can fetch call logs, filter them by specific parameters, and retrieve details for individual calls — making it straightforward to build a richer communication experience.

## Fetching Call Logs

Use `CallLogRequestBuilder` to fetch call logs. It lets you customize the request with filters like call type, status, and direction.

```javascript theme={null}
let callLogRequest = new CometChatCalls.CallLogRequestBuilder()
  .setLimit(30)
  .setAuthToken(loggedInUser.getAuthToken())
  .setCallCategory("call")
  .build();
```

### Available Settings

| Setting | Description |
| --- | --- |
| `setLimit(limit: number)` | Number of call logs to fetch. |
| `setCallType(callType: 'video' \| 'audio')` | Filter by call type. |
| `setCallStatus(callStatus: 'ongoing' \| 'busy' \| 'rejected' \| 'cancelled' \| 'ended' \| 'missed')` | Filter by call status. |
| `setHasRecording(hasRecording: boolean)` | Fetch only calls with recordings. |
| `setCallCategory(callCategory: 'call' \| 'meet')` | Filter by call category. |
| `setCallDirection(callDirection: 'incoming' \| 'outgoing')` | Filter by call direction. |
| `setUid(uid: string)` | Fetch call logs for a specific user by UID. |
| `setGuid(guid: string)` | Fetch call logs for a specific group by GUID. |
| `setAuthToken(authToken: string)` | Auth token of the logged-in user. |

### Fetch Next

Use `fetchNext()` to retrieve the next set of call logs.

```javascript theme={null}
let callLogRequest = new CometChatCalls.CallLogRequestBuilder()
  .setLimit(30)
  .setAuthToken(loggedInUser.getAuthToken())
  .setCallCategory("call")
  .build();

callLogRequest.fetchNext()
  .then(callLogHistory => {
    console.log(callLogHistory);
  })
  .catch(err => {
    console.log(err);
  });
```

### Fetch Previous

Use `fetchPrevious()` to retrieve the previous set of call logs.

```javascript theme={null}
let callLogRequest = new CometChatCalls.CallLogRequestBuilder()
  .setLimit(30)
  .setAuthToken(loggedInUser.getAuthToken())
  .setCallCategory("call")
  .build();

callLogRequest.fetchPrevious()
  .then(callLogHistory => {
    console.log(callLogHistory);
  })
  .catch(err => {
    console.log(err);
  });
```

## Get Call Details

Use `getCallDetails()` to retrieve details for a specific call. It requires the session ID and the auth token of the logged-in user.

```javascript theme={null}
const sessionID = "SESSION_ID";

CometChatCalls.getCallDetails(sessionID, authToken)
  .then((callLogs: Array<CallLog>) => {
    console.log(callLogs);
  })
  .catch(err => {
    console.log(err);
  });
```

<Note>Replace `"SESSION_ID"` with the ID of the session you want to retrieve.</Note>
