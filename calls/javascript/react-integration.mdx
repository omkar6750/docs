This guide walks you through integrating the CometChat Calls SDK into a React application. By the end, you'll have a working video call implementation with proper lifecycle handling.

## Prerequisites

Before you begin, make sure you've completed the [prerequisites](/calls/javascript/overview#prerequisites) from the Overview page. You'll also need:
- A React project

## Step 1: Install the SDK

Install the CometChat Calls SDK package:

```bash
npm install @cometchat/calls-sdk-javascript
```

## Step 2: Initialize and Login

### Initialize

Initialize the SDK with your app credentials. Call this once when your application starts.

```javascript
import { CometChatCalls } from "@cometchat/calls-sdk-javascript";

const { error } = await CometChatCalls.init({
  appId: "APP_ID",     // Replace with your App ID
  region: "REGION",    // Replace with your Region (us, eu, in)
  authKey: "AUTH_KEY", // (Optional) Replace with your Auth Key
});

if (error) {
  console.error("CometChatCalls initialization failed:", error);
} else {
  console.log("CometChatCalls initialized successfully");
}
```

### Login

Log in the user with their UID. Ensure the SDK is initialized before calling `login()`.

```javascript
CometChatCalls.login("UID") // Replace with your user's UID
  .then((user) => {
    console.log("Logged in successfully", user);
  });
```

## Step 3: Generate Token

Generate a call token for the session. The call token is required to authenticate the user for that particular session.

```javascript
CometChatCalls.generateToken("SESSION_ID") // Replace with your Session ID
  .then(({ token }) => {
    console.log("token", token);
  });
```

## Step 4: Join Session

Use the generated token to join the session along with your call settings and a container element.

```typescript
const containerRef = useRef(null);

const sessionSettings: SessionSettings = {};
CometChatCalls.joinSession(token, sessionSettings, containerRef.current);

// In your JSX
// <div ref={containerRef} style={{ width: "100vw", height: "100vh" }} />
```

For all available call settings options, see [Session Settings](/calls/javascript/session-settings).

## Step 5: Full Example

Here's a complete React component that initializes the SDK, logs in, generates a token, and joins a call session inside a `useEffect`:

```tsx
import { useEffect, useRef } from "react";
import { CometChatCalls } from "@cometchat/calls-sdk-javascript";

export default function CallScreen() {
  const containerRef = useRef(null);

  useEffect(() => {
    async function startCall() {
      // 1. Initialize
      const { error } = await CometChatCalls.init({
        appId: "APP_ID", // Replace with your App ID
        region: "REGION", // Replace with your Region (us, eu, in)
        authKey: "AUTH_KEY", // (Optional) Replace with your Auth Key
      });

      if (error) {
        console.error("CometChatCalls initialization failed:", error);
        return;
      }

      // 2. Login
      await CometChatCalls.login("UID"); // Replace with your user's UID

      // 3. Generate Token
      const { token } = await CometChatCalls.generateToken("SESSION_ID"); // Replace with your Session ID

      // 4. Join Session
      const sessionSettings: SessionSettings = {};
      CometChatCalls.joinSession(token, sessionSettings, containerRef.current);
    }

    startCall();

    return () => {
      CometChatCalls.leaveSession();
    };
  }, []);

  return (
    <div
      ref={containerRef}
      style={{ width: "100%", height: "100vh" }}
    />
  );
}
```
