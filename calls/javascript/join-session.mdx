To start a call, you need two things: a container element where the call UI renders, and a call token to authenticate the user into the session.

## Container Setup

Add a container element to your HTML:

```html
<div id="call-container" style="width: 100vw; height: 100vh;"></div>
```

## Generate Call Token

A call token authenticates a user for a specific session. Generate one using the SDK:

```javascript
const sessionId = "SESSION_ID";

try {
  const result = await CometChatCalls.generateToken(sessionId);
  console.log("Token generated:", result.token);
} catch (error) {
  console.error("Token generation failed:", error);
}
```

| Parameter | Type | Description |
|-----------|------|-------------|
| `sessionId` | String | Unique identifier for the call session |

<Note>
`generateToken()` uses the auth token of the currently logged-in user. Ensure a user is logged in before calling this method.
</Note>

## Join Session

Once you have a call token, join the session:

```typescript
const container = document.getElementById("call-container");

const sessionSettings: SessionSettings = {
  sessionType: "VIDEO",
  layout: "TILE",
  startAudioMuted: false,
  startVideoPaused: false,
};

CometChatCalls.joinSession(callToken, sessionSettings, container);
```

| Parameter | Type | Description |
|-----------|------|-------------|
| `callToken` | String | The generated call token |
| `sessionSettings` | Object | Configuration for the session (see [Session Settings](/calls/javascript/session-settings)) |
| `container` | HTMLElement | Container element for the call UI |
