The session settings object allows you to customize every aspect of your call session before participants join. From controlling the initial audio/video state to customizing the UI layout and hiding specific controls, these settings give you complete control over the call experience.

<Note>
These are pre-session configurations that must be set before joining a call. Once configured, pass the settings object to the `joinSession()` method. Settings cannot be changed after the session has started, though many features can be controlled dynamically during the call using call actions.
</Note>

```typescript
const sessionSettings: SessionSettings = {
  sessionType: "VIDEO",
  layout: "TILE",
  displayName: "",
  title: "",
  startAudioMuted: false,
  startVideoPaused: false,
  autoStartRecording: false,
  enableNoiseReduction: false,
  enableSpotlightDrag: true,
  enableSpotlightSwap: true,
  enableParticipantContextMenu: true,
  idleTimeoutPeriodBeforePrompt: 60000,
  idleTimeoutPeriodAfterPrompt: 180000,
  hideControlPanel: false,
  hideHeaderPanel: false,
  hideLeaveSessionButton: false,
  hideToggleAudioButton: false,
  hideToggleVideoButton: false,
  hideRecordingButton: true,
  hideScreenSharingButton: false,
  hideChangeLayoutButton: false,
  hideVirtualBackgroundButton: false,
  hideRaiseHandButton: false,
  hideShareInviteButton: true,
  hideParticipantListButton: false,
  hideChatButton: true,
  hideSessionTimer: false,
  hideNetworkIndicator: true,
  hideRecordingStatusIndicator: false,
  hideSwitchCameraButton: false,
};
```

## Session Settings

### Session Type

**Property:** `sessionType`

Defines the type of call session. Choose `VIDEO` for video calls with camera enabled, or `VOICE` for audio-only calls.

```javascript
sessionType: "VIDEO"
```

| Value | Description |
|-------|-------------|
| `VIDEO` | Video call with camera enabled |
| `VOICE` | Audio-only call |

**Default:** `VIDEO`

### Layout Mode

**Property:** `layout`

Sets the initial layout mode for displaying participants.

```javascript
layout: "TILE"
```

| Value | Description |
|-------|-------------|
| `TILE` | Grid layout showing all participants equally |
| `SIDEBAR` | Main speaker with participants in a sidebar |
| `SPOTLIGHT` | Focus on active speaker/shared-screen with others in sidebar |

**Default:** `TILE`

### Start Audio Muted

**Property:** `startAudioMuted`

Determines whether the microphone is muted when joining the session.

```javascript
startAudioMuted: true
```

**Default:** `false`

### Start Video Paused

**Property:** `startVideoPaused`

Controls whether the camera is turned off when joining the session.

```javascript
startVideoPaused: true
```

**Default:** `false`

### Auto Start Recording

**Property:** `autoStartRecording`

Automatically starts recording the session as soon as it begins.

```javascript
autoStartRecording: true
```

**Default:** `false`

### Idle Timeout Period Before Prompt

**Property:** `idleTimeoutPeriodBeforePrompt`

Time in milliseconds before showing the idle timeout prompt when you're the only participant.

```javascript
idleTimeoutPeriodBeforePrompt: 60000 // 60 seconds
```

**Default:** `60000` (60 seconds)

### Idle Timeout Period After Prompt

**Property:** `idleTimeoutPeriodAfterPrompt`

Time in milliseconds after the prompt before automatically ending the session.

```javascript
idleTimeoutPeriodAfterPrompt: 180000 // 180 seconds
```

**Default:** `180000` (180 seconds)

### Display Name

**Property:** `displayName`

Sets the display name shown for the local participant in the call.

```javascript
displayName: "John"
```

**Default:** `""` (empty string)

### Title

**Property:** `title`

Sets a title displayed in the call header UI, such as a meeting or session title.

```javascript
title: "Team Standup"
```

**Default:** `""` (empty string)

### Initial Camera Facing

**Property:** `initialCameraFacing`

Sets the initial camera facing direction on supported mobile devices. Use `FRONT` for the front-facing camera or `BACK` for the rear camera.

```javascript
initialCameraFacing: "FRONT"
```

| Value | Description |
|-------|-------------|
| `FRONT` | Front-facing camera |
| `BACK` | Rear camera |

**Default:** `undefined` (uses device default)

### Enable Noise Reduction

**Property:** `enableNoiseReduction`

Enables algorithmic noise reduction on the audio stream to filter out background noise.

```javascript
enableNoiseReduction: true
```

**Default:** `false`

### Enable Spotlight Drag

**Property:** `enableSpotlightDrag`

Allows participants to drag and reposition the spotlight (main video) within the call layout.

```javascript
enableSpotlightDrag: true
```

**Default:** `true`

### Enable Spotlight Swap

**Property:** `enableSpotlightSwap`

Allows participants to swap which participant is in the spotlight by clicking on another participant's video.

```javascript
enableSpotlightSwap: true
```

**Default:** `true`

### Enable Participant Context Menu

**Property:** `enableParticipantContextMenu`

Enables the context menu (three-dot more button) on each participant's video tile, providing quick access to actions like mute, pin, or remove.

```javascript
enableParticipantContextMenu: true
```

**Default:** `true`

### Audio Input Device ID

**Property:** `audioInputDeviceId`

Pre-selects a specific audio input (microphone) device by its device ID. When not set, the system default device is used.

```javascript
audioInputDeviceId: "device-id-string"
```

**Default:** `undefined` (uses system default)

### Audio Output Device ID

**Property:** `audioOutputDeviceId`

Pre-selects a specific audio output (speaker) device by its device ID. When not set, the system default device is used.

```javascript
audioOutputDeviceId: "device-id-string"
```

**Default:** `undefined` (uses system default)

### Video Input Device ID

**Property:** `videoInputDeviceId`

Pre-selects a specific video input (camera) device by its device ID. When not set, the system default device is used.

```javascript
videoInputDeviceId: "device-id-string"
```

**Default:** `undefined` (uses system default)

## UI Visibility Settings

### Hide Control Panel

**Property:** `hideControlPanel`

Hides the bottom control bar that contains call action buttons.

```javascript
hideControlPanel: true
```

**Default:** `false`

### Hide Leave Session Button

**Property:** `hideLeaveSessionButton`

Hides the button that allows users to leave the session.

```javascript
hideLeaveSessionButton: true
```

**Default:** `false`

### Hide Toggle Audio Button

**Property:** `hideToggleAudioButton`

Hides the microphone mute/unmute button from the control panel.

```javascript
hideToggleAudioButton: true
```

**Default:** `false`

### Hide Toggle Video Button

**Property:** `hideToggleVideoButton`

Hides the camera on/off button from the control panel.

```javascript
hideToggleVideoButton: true
```

**Default:** `false`

### Hide Recording Button

**Property:** `hideRecordingButton`

Hides the recording start/stop button from the control panel.

```javascript
hideRecordingButton: false
```

**Default:** `true`

### Hide Screen Sharing Button

**Property:** `hideScreenSharingButton`

Hides the screen sharing button from the control panel.

```javascript
hideScreenSharingButton: true
```

**Default:** `false`

### Hide Change Layout Button

**Property:** `hideChangeLayoutButton`

Hides the button that allows switching between different layout modes.

```javascript
hideChangeLayoutButton: true
```

**Default:** `false`

### Hide Virtual Background Button

**Property:** `hideVirtualBackgroundButton`

Hides the virtual background settings button.

```javascript
hideVirtualBackgroundButton: true
```

**Default:** `false`

### Hide Network Indicator

**Property:** `hideNetworkIndicator`

Hides the network quality indicator.

```javascript
hideNetworkIndicator: false
```

**Default:** `true`

### Hide Header Panel

**Property:** `hideHeaderPanel`

Hides the top header panel of the call UI.

```javascript
hideHeaderPanel: true
```

**Default:** `false`

### Hide Raise Hand Button

**Property:** `hideRaiseHandButton`

Hides the raise hand button from the control panel.

```javascript
hideRaiseHandButton: true
```

**Default:** `false`

### Hide Share Invite Button

**Property:** `hideShareInviteButton`

Hides the button that allows participants to share an invite link to the session.

```javascript
hideShareInviteButton: false
```

**Default:** `true`

### Hide Participant List Button

**Property:** `hideParticipantListButton`

Hides the button that opens the participant list panel.

```javascript
hideParticipantListButton: true
```

**Default:** `false`

### Hide Chat Button

**Property:** `hideChatButton`

Hides the in-call chat button from the control panel.

```javascript
hideChatButton: false
```

**Default:** `true`

### Hide Session Timer

**Property:** `hideSessionTimer`

Hides the call duration timer displayed in the UI.

```javascript
hideSessionTimer: true
```

**Default:** `false`

### Hide Recording Status Indicator

**Property:** `hideRecordingStatusIndicator`

Hides the recording status badge that blinks a red light when a recording is in progress.

```javascript
hideRecordingStatusIndicator: true
```

**Default:** `false`

### Hide Switch Camera Button

**Property:** `hideSwitchCameraButton`

Hides the button that allows switching between front and back cameras on supported mobile devices.

```javascript
hideSwitchCameraButton: true
```

**Default:** `false`

## Complete Example

```typescript
const sessionSettings: SessionSettings = {
  // Session configuration
  sessionType: "VIDEO",
  layout: "TILE",
  displayName: "",
  title: "",
  startAudioMuted: false,
  startVideoPaused: false,
  autoStartRecording: false,

  // Feature settings
  enableNoiseReduction: false,
  enableSpotlightDrag: true,
  enableSpotlightSwap: true,
  enableParticipantContextMenu: true,
  initialCameraFacing: "FRONT",

  // Timeout settings
  idleTimeoutPeriodBeforePrompt: 60000,
  idleTimeoutPeriodAfterPrompt: 180000,

  // Device selection
  audioInputDeviceId: undefined,
  audioOutputDeviceId: undefined,
  videoInputDeviceId: undefined,

  // UI visibility
  hideControlPanel: false,
  hideHeaderPanel: false,
  hideLeaveSessionButton: false,
  hideToggleAudioButton: false,
  hideToggleVideoButton: false,
  hideRecordingButton: true,
  hideScreenSharingButton: false,
  hideChangeLayoutButton: false,
  hideVirtualBackgroundButton: false,
  hideRaiseHandButton: false,
  hideShareInviteButton: true,
  hideParticipantListButton: false,
  hideChatButton: true,
  hideSessionTimer: false,
  hideNetworkIndicator: true,
  hideRecordingStatusIndicator: false,
  hideSwitchCameraButton: false,
};
```
