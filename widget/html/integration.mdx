You can embed the CometChat Widget on any HTML page using a small snippet. Each option below shows:

1. **When to use it**  
2. **Code to add inside your `<head>`**  
3. **Code to add inside your `<body>`** where the widget mounts

In every snippet, edit only the text inside quotes such as `"YOUR_APP_ID"`.

> **Hosted helper script:** Host `cometchat-auth.js` yourself and reference it over HTTPS. The examples below use the placeholder link `https://cdn.example.com/cometchat/cometchat-auth.js`.

### Register on CometChat & Gather Your Keys
Before you begin, sign up at the [CometChat Dashboard](https://app.cometchat.com/) and create a new app. Copy:
- **App ID**  
- **Region**  
- **Auth Key**

## 1. Guest / Anonymous Users (No Login)

**Use this when:**

- Any visitor should start chatting immediately  
- You do not maintain user accounts or login

### 1.1 Code for `<head>`

```html
<!-- CometChat Widget script (required for all modes) -->
<script defer src="https://cdn.jsdelivr.net/npm/@cometchat/chat-embed@1.x.x/dist/main.js"></script>
```

### 1.2 Code for `<body>`

```html
<!-- Where the chat widget should appear -->
<div id="cometChatMount" style="height:600px;"></div>

<script type="module">
  import { CometChatAuth } from "https://cdn.example.com/cometchat/cometchat-auth.js";

  //  EDIT ONLY THE TEXT INSIDE QUOTES
  const COMETCHAT_WIDGET_CONFIG = {
    appID: "YOUR_APP_ID",
    region: "YOUR_APP_REGION",   // e.g. "us", "eu", "in"
    mode: "guest",               // keep this as "guest"
    authKey: "YOUR_AUTH_KEY",

    // Optional: how the guest appears in chat
    user: {
      name: "Guest User",
      avatar: "",                // e.g. "https://example.com/guest.png"
      link: ""                   // e.g. "https://example.com/profile"
    },

    // Widget placement + size
    mount: "#cometChatMount",
    width: "100%",
    height: "100%",
    isDocked: true
  };

  CometChatAuth.start(COMETCHAT_WIDGET_CONFIG);
</script>
```

## 2. Create User on the Fly (UID Login)

**Use this when:**

- You already have unique user IDs  
- Each visitor should appear consistently in chat

### 2.1 Code for `<head>`

```html
<!-- CometChat Widget script (required for all modes) -->
<script defer src="https://cdn.jsdelivr.net/npm/@cometchat/chat-embed@1.x.x/dist/main.js"></script>
```

### 2.2 Code for `<body>`

```html
<!-- Where the chat widget should appear -->
<div id="cometChatMount" style="height:600px;"></div>

<script type="module">
  import { CometChatAuth } from "https://cdn.example.com/cometchat/cometchat-auth.js";

  //  EDIT ONLY THE TEXT INSIDE QUOTES
  const COMETCHAT_WIDGET_CONFIG = {
    appID: "YOUR_APP_ID",
    region: "YOUR_APP_REGION",      // e.g. "us", "eu", "in"
    mode: "uid",                    // keep this as "uid"
    authKey: "YOUR_AUTH_KEY",

    // IMPORTANT: must be unique per user
    uid: "UNIQUE_USER_ID",          // e.g. "user_123"

    // Optional: user profile details
    user: {
      name: "User Name",
      avatar: "https://example.com/user.png",
      link: "https://example.com/profile"
    },

    // Widget placement + size
    mount: "#cometChatMount",
    width: "100%",
    height: "100%",
    isDocked: true
  };

  CometChatAuth.start(COMETCHAT_WIDGET_CONFIG);
</script>
```

## 3. App With Its Own Backend (Auth Token Login)

**Use this when:**

- Your app already authenticates users on a backend  
- Your server issues CometChat auth tokens

### 3.1 Code for `<head>`

```html
<!-- CometChat Widget script (required for all modes) -->
<script defer src="https://cdn.jsdelivr.net/npm/@cometchat/chat-embed@1.x.x/dist/main.js"></script>
```

### 3.2 Code for `<body>`

```html
<!-- Where the chat widget should appear -->
<div id="cometChatMount" style="height:600px;"></div>

<script type="module">
  import { CometChatAuth } from "https://cdn.example.com/cometchat/cometchat-auth.js";

  //  EDIT ONLY THE TEXT INSIDE QUOTES
  const COMETCHAT_WIDGET_CONFIG = {
    appID: "YOUR_APP_ID",
    region: "YOUR_APP_REGION",       // e.g. "us", "eu", "in"
    mode: "authToken",               // keep this as "authToken"

    // Generated BY YOUR BACKEND
    authToken: "USER_AUTH_TOKEN",

    // Widget placement + size
    mount: "#cometChatMount",
    width: "100%",
    height: "100%",
    isDocked: true
  };

  CometChatAuth.start(COMETCHAT_WIDGET_CONFIG);
</script>
```
