Properties and constraints for the BYO Agent APIs.

## BYO Agents

| **Property**               | **Constraints**            |
| -------------------------- | -------------------------- |
| Agent name                 | Max 100 characters         |
| Agent URL                  | Must be a valid HTTPS URL  |
| Maximum BYO agents per app | Contact support for limits |

## Tools

| **Property**                | **Constraints**            |
| --------------------------- | -------------------------- |
| Tool name                   | Max 100 characters         |
| Maximum tools per BYO agent | Contact support for limits |
