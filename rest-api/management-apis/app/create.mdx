---
openapi: post /apps
description: "Creates an app in the account. It returns all the app related information."
---

## Constraints

| Item                          | Constraint               | Notes                             |
| ----------------------------- | ------------------------ | --------------------------------- |
| App name                      | 100 characters (UTF8mb4) | Supports all languages and emojis |
| App version                   | v3                       | Only option available             |
| Maximum collaborators per app | 25                       | Team management limit             |

For the complete error reference, see [Error Guide](/rest-api/management-apis/error-codes).
