## Constraints

| Item | Constraint | Notes |
| ---- | ---------- | ----- |
| Maximum users in a call | 50 | Supports both one-on-one and group calling; works best with 50 participants |
| Default video frame rate | 30 FPS | May adapt based on network conditions |
| Video resolution | 180p - 720p | Depends on layout selected and user bandwidth |
| Media encryption | SRTP | Ensures secure real-time communication |
| Audio codec | OPUS | Optimized for low-latency voice communication |
| Video codec | H.264 | Widely supported for cross-platform compatibility |
