## Functionality

Props that control component behavior, grouped by surface area.

### Data

| Property | Type | Default | Effect | Pitfall |
| --- | --- | --- | --- | --- |
| `call` | `CometChat.Call` | Auto-detected | Sets a specific call object for the incoming call display | Component auto-detects incoming calls if not provided |

### Sound

| Property | Type | Default | Effect | Pitfall |
| --- | --- | --- | --- | --- |
| `disableSoundForCalls` | `boolean` | `false` | Disables the incoming call ringtone | — |
| `customSoundForCalls` | `string` | `undefined` | Custom sound file URL for incoming calls | — |

## Actions

[Actions](/ui-kit/react/components-overview#actions) split into predefined (built-in triggers) and user-defined (callbacks).

### Predefined Actions

The accept button starts the call session. The decline button rejects the call. Both can be overridden.

### User-Defined Actions

#### onAccept

Fires when the accept button is clicked.

| Detail | Value |
| --- | --- |
| When it fires | User clicks the accept button |
| Payload type | `(call: CometChat.Call) => void` |
| What devs typically do next | Custom call acceptance logic |

```ts lines
import { CometChatIncomingCall } from "@cometchat/chat-uikit-react";

const IncomingCallsDemo = () => {
  const handleOnAccept = () => {
    console.log("custom on accept action");
  };

  return <CometChatIncomingCall onAccept={handleOnAccept} />;
};
```

#### onDecline

Fires when the decline button is clicked.

| Detail | Value |
| --- | --- |
| When it fires | User clicks the decline button |
| Payload type | `(call: CometChat.Call) => void` |
| What devs typically do next | Custom call rejection logic |

```ts lines
import { CometChatIncomingCall } from "@cometchat/chat-uikit-react";

const IncomingCallsDemo = () => {
  const handleOnDecline = () => {
    console.log("your custom on decline action");
  };

  return <CometChatIncomingCall onDecline={handleOnDecline} />;
};
```

#### onError

Fires when the component encounters an error.

| Detail | Value |
| --- | --- |
| When it fires | Any internal error |
| Payload type | `(error: CometChat.CometChatException) => void` |
| What devs typically do next | Log to error tracking service |

```ts lines
import { CometChatIncomingCall } from "@cometchat/chat-uikit-react";

const IncomingCallsDemo = () => {
  const handleOnError = (error: CometChat.CometChatException) => {
    console.log("your custom on error action", error);
  };

  return <CometChatIncomingCall onError={handleOnError} />;
};
```

## Filters

Set `CallSettingsBuilder` to customize the calling experience. Refer to [CallSettingsBuilder](/sdk/javascript/direct-call#settings).

```ts lines
import { CometChatIncomingCall } from "@cometchat/chat-uikit-react";

const IncomingCallsDemo = () => {
  const getCallSettingsBuilder = (call: CometChat.Call) => {
    return new CometChatCalls.CallSettingsBuilder()
      .setIsAudioOnlyCall(
        call?.getType() === CometChatUIKitConstants.MessageTypes.audio
      )
      .build();
  };

  return <CometChatIncomingCall callSettingsBuilder={getCallSettingsBuilder} />;
};
```

## Events

### UI Events

Events emitted by the IncomingCall component:

| Event | Source | When it fires |
| --- | --- | --- |
| `ccCallRejected` | `CometChatCallEvents` | Incoming call is rejected |
| `ccCallAccepted` | `CometChatCallEvents` | Incoming call is accepted |
| `ccCallEnded` | `CometChatCallEvents` | Call ends |

#### Subscribe

```tsx lines
const ccCallRejected = CometChatCallEvents.ccCallRejected.subscribe(
  (call: CometChat.Call) => {
    //Your Code
  }
);

const ccCallAccepted = CometChatCallEvents.ccCallAccepted.subscribe(
  (call: CometChat.Call) => {
    //Your Code
  }
);

const ccCallEnded = CometChatCallEvents.ccCallEnded.subscribe(
  (call: CometChat.Call) => {
    //Your Code
  }
);
```

#### Unsubscribe

```tsx lines
ccCallRejected?.unsubscribe();
ccCallAccepted?.unsubscribe();
ccCallEnded?.unsubscribe();
```

### SDK Events (Real-Time)

The component internally listens to SDK call events for real-time incoming call detection. No manual attachment needed.
