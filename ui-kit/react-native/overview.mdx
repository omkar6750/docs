<Info>
**Quick Reference for AI Agents & Developers**

```bash
# Install the UI Kit and the required Chat SDK
npm i @cometchat/chat-uikit-react-native
npm i @cometchat/chat-sdk-react-native
```

```tsx
import { CometChatUIKit, UIKitSettings } from "@cometchat/chat-uikit-react-native";
import { CometChat } from "@cometchat/chat-sdk-react-native";

// Initialize
const uiKitSettings: UIKitSettings = {
  appId: "APP_ID",
  authKey: "AUTH_KEY",
  region: "us",
  subscriptionType: CometChat.AppSettings.SUBSCRIPTION_TYPE_ALL_USERS as UIKitSettings["subscriptionType"],
};
await CometChatUIKit.init(uiKitSettings);

// Login
await CometChatUIKit.login({ uid: "cometchat-uid-1" });
```

With CometChat's UI Kit for React Native, you can effortlessly build a chat app equipped with all the essential messaging features, along with customizable options tailored to your application requirements. This UI Kit comprises prebuilt UI components organized into smaller modules and components, each configurable to meet your specific needs.

<Warning>
You must call `CometChatUIKit.init(UIKitSettings)` before rendering any UI Kit components or calling any SDK methods. Initialization must complete before login.
</Warning>

<Warning>
**Auth Key** is for development/testing only. In production, generate **Auth Tokens** on your server using the REST API and pass them to the client. Never expose Auth Keys in production client code.
</Warning>

## Before Getting Started

Before you begin, it's essential to grasp the fundamental concepts and features offered by CometChat's APIs, SDK, and UI Kit. You can find detailed information in the [Key Concepts](/fundamentals/key-concepts) documentation.

The UI Kit library comprises pre-built React Native components for effortless integration and is built on top of the [React Native Chat SDK](/sdk/react-native/overview). You need to install the Chat SDK alongside the UI Kit as a required dependency.

To begin, follow the [Getting Started](/ui-kit/react-native/getting-started) guide.
