CometChat integrates with popular third-party platforms to provide unified customer communication experiences. These integrations allow you to connect your chat with external support and messaging systems.

<Note>
**Available via:** [UI Kits](/ui-kit/flutter/overview) | [SDK](/sdk/flutter/overview) | [Dashboard](https://app.cometchat.com)
</Note>

***

## Chatwoot

Connect and exchange messages between CometChat and Chatwoot for unified customer support. This integration enables seamless communication between your app users and support agents using Chatwoot.

| Feature | Description |
| --- | --- |
| **Bi-directional Messaging** | Messages sync between CometChat and Chatwoot |
| **Agent Assignment** | Route conversations to appropriate agents |
| **Conversation History** | Maintain full conversation context |

For detailed setup and configuration, see [Chatwoot Integration](/fundamentals/chatwoot).

***

## Intercom

Connect and exchange messages between CometChat and Intercom for seamless customer communication. This integration bridges your in-app chat with Intercom's powerful customer engagement platform.

| Feature | Description |
| --- | --- |
| **Message Sync** | Real-time message synchronization |
| **User Context** | Share user data between platforms |
| **Support Workflows** | Leverage Intercom's support features |

For detailed setup and configuration, see [Intercom Integration](/fundamentals/intercom).

***

## Setting Up Integrations

Third-party integrations are configured through the CometChat Dashboard:

1. Navigate to your app in the [CometChat Dashboard](https://app.cometchat.com)
2. Go to **Extensions** section
3. Find and enable the desired integration (Chatwoot or Intercom)
4. Configure the integration settings with your third-party credentials

<Note>
Each integration requires an active account with the respective third-party service. Refer to the specific integration documentation for detailed setup instructions.
</Note>
