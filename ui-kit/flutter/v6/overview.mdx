The **CometChat UI Kit V6** for Flutter is a major architectural evolution of the Flutter Chat UIKit. It provides the same robust set of **prebuilt UI widgets** that are **modular, customizable, and highly scalable**, now built on **clean architecture** with **BLoC state management** for better testability, maintainability, and performance.

***

## **What's New in V6?**

* **BLoC State Management** ‚Äì Replaces GetX with flutter_bloc for predictable, testable state.
* **Clean Architecture** ‚Äì Every module follows `bloc/` + `data/` + `domain/` + `di/` layers.
* **No DataSource Decorator Chain** ‚Äì Direct static method calls via `MessageTemplateUtils` replace the 10-layer decorator pattern.
* **Internalized Shared UI** ‚Äì `cometchat_uikit_shared` is now part of the package, no separate dependency needed.
* **Faster Startup** ‚Äì No extension registration or network calls at init time.

***

## **Why Choose CometChat UI Kit V6?**

* **Rapid Integration** ‚Äì Prebuilt UI widgets for faster deployment.
* **Customizable & Flexible** ‚Äì Modify the UI to align with your brand's identity.
* **Cross-Platform Compatibility** ‚Äì Works seamlessly across iOS and Android platforms.
* **Scalable & Reliable** ‚Äì Built on CometChat's robust chat infrastructure.
* **Testable** ‚Äì BLoC pattern with `bloc_test` and `mocktail` for comprehensive testing.
* **Better Performance** ‚Äì ~500-2000ms faster startup, no decorator chain overhead.

***

## **Integration**

### **UI Components (Assemble It Yourself)**

A collection of individual widgets‚Äîlike conversation lists, message lists, message composer, etc.‚Äîeach with built-in chat logic so you can customize every element.

**How It Works**

* Import the widgets you need from the UI Kit.
* Arrange them in your desired layout, applying styling or customization as needed.
* Each widget integrates with CometChat logic via BLoC and clean architecture layers.

**Why It's Great**

* **Flexible Design** ‚Äì You control the final UI arrangement.
* **No Extra Overhead** ‚Äì Implement only the features you need.
* **Modular** ‚Äì Use exactly what you want, when you want.
