The **Android Calls SDK** enables real‑time voice & video (1:1 and group) with a minimal, flexible API. This guide follows a step‑by‑step Getting Started, adapted for media sessions instead of chat UI components.

<Info>
You should have: (1) App ID (2) Region (3) Auth Key or Auth Token. Create / manage these in the <a href="https://app.cometchat.com" target="_blank">CometChat Dashboard</a>.
</Info>

---

## **Prerequisites**

| Requirement | Notes |
|-------------|-------|
| Min API Level | 24 (Calls) / 21 (app baseline) |
| Java / Kotlin | Java 8 (source & target) |
| Internet | Stable network for low‑latency media |
| Hardware | Camera + Mic (emulator limited) |

<Info>
The Calls SDK depends on the Core Chat SDK for authentication context. Always initialize & login with the Chat SDK first.
</Info>

---

## **Step 1: Add Repository**

Add the CometChat Maven repository in `settings.gradle` (preferred modern approach) or project `build.gradle`.

```gradle settings.gradle
dependencyResolutionManagement {
	repositoriesMode.set(RepositoriesMode.FAIL_ON_PROJECT_REPOS)
	repositories {
		google()
		mavenCentral()
		maven { url "https://dl.cloudsmith.io/public/cometchat/cometchat/maven/" }
	}
}
```

## **Step 3: Permissions**

```xml AndroidManifest.xml
<uses-permission android:name="android.permission.INTERNET" />
<uses-permission android:name="android.permission.RECORD_AUDIO" />
<uses-permission android:name="android.permission.CAMERA" />
<uses-permission android:name="android.permission.MODIFY_AUDIO_SETTINGS" />
<uses-permission android:name="android.permission.BLUETOOTH" />
<uses-permission android:name="android.permission.BLUETOOTH_CONNECT" />
```

Request CAMERA & RECORD_AUDIO at runtime just‑in‑time (first call attempt) to improve accept rates.

## **Step 4: Initialize Core Chat + Calls SDK**

<Tabs>
<Tab title="Java">

```java MainApplication.java (or Splash Activity)
String appID = "APP_ID"; // Replace
String region = "REGION"; // us / eu

AppSettings appSettings = new AppSettings.AppSettingsBuilder()
		.subscribePresenceForAllUsers()
		.setRegion(region)
		.autoEstablishSocketConnection(true)
		.build();

CometChat.init(this, appID, appSettings, new CometChat.CallbackListener<String>() {
	@Override public void onSuccess(String s) { }
	@Override public void onError(CometChatException e) { }
});

CallAppSettings callAppSettings = new CallAppSettingBuilder()
		.setAppId(appID)
		.setRegion(region)
		.build();

CometChatCalls.init(this, callAppSettings, new CometChatCalls.CallbackListener<String>() {
	@Override public void onSuccess(String s) { }
	@Override public void onError(CometChatException e) { }
});
```

</Tab>

## **Step 6: Generate Call Token (Direct Calling)**

<Tabs>
<Tab title="Java">

```java MainActivity.java (snippet)
String sessionId = UUID.randomUUID().toString();
String userAuthToken = CometChat.getUserAuthToken();

CometChatCalls.generateToken(sessionId, userAuthToken,
	new CometChatCalls.CallbackListener<GenerateToken>() {
		@Override public void onSuccess(GenerateToken t) {
			startCall(t.getToken());
		}
		@Override public void onError(CometChatException e) { }
});
```

</Tab>
<Tab title="Kotlin">

```kotlin MainActivity.kt (snippet)
val sessionId = UUID.randomUUID().toString()
val authToken = CometChat.getUserAuthToken()

CometChatCalls.generateToken(sessionId, authToken,
	object: CometChatCalls.CallbackListener<GenerateToken>() {
		override fun onSuccess(t: GenerateToken) { startCall(t.token) }
		override fun onError(e: CometChatException) { }
	}
)
```

</Tab>
</Tabs>

<Info>
Default (invite/accept) calling flow: obtain `sessionId` from the incoming `Call` object instead of generating a random one.
</Info>

## **Step 7: Start Session & Render Video**

Layout container:

```xml activity_main.xml
<RelativeLayout
	android:id="@+id/videoContainer"
	android:layout_width="match_parent"
	android:layout_height="match_parent" />
```

<Tabs>
<Tab title="Java">

```java MainActivity.java (session)
RelativeLayout videoContainer = findViewById(R.id.videoContainer);
CallSettings callSettings = new CometChatCalls.CallSettingsBuilder(this, videoContainer)
	.setIsAudioOnly(false)
	.setDefaultLayoutEnable(true)
	.setEventListener(new CometChatCallsEventsListener() {
		@Override public void onUserJoined(RTCUser user) {}
		@Override public void onUserLeft(RTCUser user) {}
		@Override public void onCallEnded() { CometChatCalls.endSession(); }
		@Override public void onCallEndButtonPressed() {}
		@Override public void onRecordingToggled(RTCRecordingInfo info) {}
		@Override public void onAudioModeChanged(ArrayList<AudioMode> devices) {}
		@Override public void onError(CometChatException e) {}
	})
	.build();

CometChatCalls.startSession(callToken, callSettings, new CometChatCalls.CallbackListener<String>() {
	@Override public void onSuccess(String s) { }
	@Override public void onError(CometChatException e) { }
});
```

</Tab>
<Tab title="Kotlin">


## **Step 10: Error Codes (Common)**

| Code | Meaning |
|------|---------|
| INVALID_ARGUMENT_EXCEPTION | Missing / invalid parameters |
| AUTH_ERR_TOKEN_INVALID_SIGNATURE | Token invalid / expired |
| API_ERROR | Server returned an error |
| UNKNOWN_API_ERROR | Network / connectivity issue |

## **Troubleshooting**

| Issue | Fix |
|-------|-----|
| Black tiles | Check camera permission / physical camera |
| No video | Ensure `.setIsAudioOnly(false)` & not paused |
| No events | Listener must be set before `startSession()` |
| Token errors | Regenerate & confirm correct Region |
| Audio route wrong | Call `setAudioMode()` after session start |
